BackCountry Meal Planner

Backcountry Meal Planner streamlines meal packing and planning for backcountry trips. It returns detailed nutrition info, packing lists and amounts, and dietary restriction information, which saves time packing. Built using Python, Flask, PostgreSQL, and the Food Data Central API.

Employee Scheduler

Employee Scheduler is an application for organizations to manage contracts and short term work agreements for staff. Organization administrators can assign staff to jobs, keep track of which jobs/tasks still need to be filled, and view all the jobs on a calendar. Built using React, Node.js, PostgreSQL, and the Microsoft Graph API.
